# 🛡️ Server Security Toolkit

Модульный инструмент для автоматизации настройки безопасности Ubuntu серверов.

## 📋 Возможности

- 🔐 **SSH Security**: Смена портов, отключение паролей, настройка ключей
- 🛡️ **Firewall Setup**: Автоматическая настройка UFW
- 🔧 **System Hardening**: Укрепление системы безопасности
- 🔑 **SSH Key Management**: Генерация и управление ключами
- 📊 **Logging**: Подробное логирование всех операций
- 🧪 **Test Mode**: Безопасное тестирование без изменения системы

## 🏗️ Структура проекта

server-security-toolkit/
├── main.sh # Главный скрипт
├── modules/ # Модули функциональности
│ ├── ssh_security.sh # SSH безопасность
│ ├── firewall.sh # Настройка файрвола
│ └── system_hardening.sh # Укрепление системы
├── configs/ # Шаблоны конфигураций
├── keys/ # SSH ключи
├── logs/ # Файлы логов
├── scripts/ # Вспомогательные скрипты
├── tests/ # Тестовые скрипты
└── .vscode/ # Настройки VS Code


## 🚀 Использование

Клонирование репозитория
git clone https://github.com/USERNAME/server-security-toolkit.git
cd server-security-toolkit

Сделать исполняемым
chmod +x main.sh

Тестовый режим (безопасно)
./main.sh

Выберите опцию 8
Продакшн режим (требует sudo)
sudo ./main.sh


## 📦 Системные требования

- Ubuntu 18.04+ (тестировался на 22.04 LTS)
- Bash 4.0+
- Права администратора
- Пакеты: ssh, ufw, systemctl
